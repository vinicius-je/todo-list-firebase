# todo-list-firebase

## Objetivo do projeto:
Criar um todo list com firebase

## Tópicos trabalhados:
* Firebase Authentication
* Cloud Firestore

## Implementação:
- [x] Criar uma conta
- [x] Deletar uma conta
- [x] Acessar a conta (sign in)
- [x] Sair da conta (sign out)
- [x] Acessar a lista de tarefa
- [x] Marcar tarefa como concluída
- [x] Excluir a tarefa
